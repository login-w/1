
## 这是我的个人博客系统   
### 使用的技术栈为：  


#### 后端：
* 核心框架：springboot
* jdk版本：jdk1.8
* jar包管理：maven3.3.9
* 数据操作模板：thymeleaf
* 分页插件：pageHelper
#### 前端：
* js框架：jquery
* css框架：semantic-ui
* 编辑器：markdown文本编辑器
* 代码高亮插件：prism插件
* 动态效果插件：animate
* 生成文章目录结构的插件：tocbot
* 生成二维码插件：qrcode
#### 数据库：
* mysql：5.6
* 数据库框架：mybatis
